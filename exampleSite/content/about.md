OhGoshTheTheme is a Hugo theme inspired by retro computer terminals. It is designed to be minimal, fast, and accessible.

## Features

- **Minimal Design**: Clean, uncluttered layout
- **Multiple Color Schemes**: Switch between 5 built-in themes
- **Responsive**: Looks great on all devices
- **Dark Mode**: Built-in dark/light theme toggle
- **Search**: Client-side search powered by Fuse.js
- **Syntax Highlighting**: Code blocks with Chroma
- **Social Icons**: Link to your social profiles
- **Archive**: Browse posts by date
- **Profile Mode**: Use as a personal landing page
